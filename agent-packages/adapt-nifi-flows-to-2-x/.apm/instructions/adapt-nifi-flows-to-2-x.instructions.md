## Skill trigger: `adapt-nifi-flows-to-2-x`

When working with an `upgradeAdvisorReport.csv` produced by the NiFi upgradeAdvisor script, or when the user asks to
apply NiFi 1.x to 2.x upgrade recommendations to exported flow JSON files, invoke the `adapt-nifi-flows-to-2-x` skill.
